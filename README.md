# Holidaze
Escape the everyday and discover your perfect holiday haven! 

This front-end project is a modern travel booking platform. Users can browse and book vacation rentals, unique stays, and experiences. Hosts can list their properties, set availability, prices, and manage bookings. The application focuses on responsive UI, smooth user flows, intuitive search & filters, and clean design using React, TypeScript, Vite, and Tailwind CSS.
